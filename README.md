# HTML Resume Page

## Description
This project is a single-page resume website built using **only HTML**, as per the assignment instructions.  
No CSS or JavaScript has been used. The focus is on clean structure, proper HTML semantics, and readability.

## Features
- Semantic HTML structure
- Clear section separation
- Skills listed using unordered lists
- Experience and education presented using tables
- Clean and readable layout without CSS

## Technologies Used
- HTML5

## Setup Instructions
1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

3. Open `index.html` in any modern web browser.

## Usage
- Open the `index.html` file directly in a browser to view the resume.
- The file can be hosted using GitHub Pages or any static hosting service.
